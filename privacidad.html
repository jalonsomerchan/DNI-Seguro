PRIVACIDAD COMPROBABLE

Tu documento se queda en tu dispositivo

DNI Seguro procesa cada imagen en la memoria de tu navegador. No crea una cuenta, no guarda copias y no envía el documento a una API o servidor de procesamiento.

Qué ocurre al usar la herramienta

  1. El navegador lee el archivo que eliges mediante APIs locales.
  2. La censura, la marca de agua y la composición se dibujan en un elemento Canvas en tu dispositivo.
  3. El modo OCR analiza el documento localmente. La primera vez puede descargar los archivos del motor OCR, pero no envía tu imagen con esa descarga.
  4. El resultado se genera como un archivo nuevo que tú decides descargar o compartir.

Las imágenes solo viven durante la sesión. Al recargar o cerrar la pestaña se eliminan de la memoria de la aplicación.

Cómo comprobar que no se sube nada

Revisa el código fuente

El proyecto es open source y está publicado en jalonsomerchan/dni-seguro. Puedes buscar llamadas de red como fetch, XMLHttpRequest o formularios: el flujo de documentos no utiliza ninguna de ellas.

Observa la pestaña Red

  1. Abre las herramientas para desarrolladores del navegador.
  2. Entra en la pestaña Red y limpia la lista.
  3. Carga un documento, censúralo y descárgalo.
  4. Comprueba que no existe una petición cuyo cuerpo contenga la imagen. En OCR sí pueden aparecer descargas de JavaScript, WebAssembly o datos de idioma.

Usa el modo offline

Tras visitar o instalar la webapp, desconecta la red y usa la Versión Simple. Que la edición siga funcionando demuestra que no depende de un servidor para procesar la imagen.

¿Por qué importa?

Un DNI contiene identificadores permanentes, firma, fotografía y otros datos con valor para la suplantación. Evitar una copia adicional en un servidor reduce la superficie de exposición, las filtraciones accidentales y la dependencia de la política de retención de un tercero.

Límites y buenas prácticas

  • Revisa siempre la imagen final: el OCR puede equivocarse.
  • Oculta solo los datos que el destinatario no necesite.
  • Añade una marca de agua con destinatario, finalidad y fecha.
  • Usa un dispositivo y navegador de confianza.
  • DNI Seguro protege una copia, pero no garantiza que el destinatario haga un uso correcto de ella.
